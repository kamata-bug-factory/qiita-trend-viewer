# Qiita Trend Viewer 仕様書

## 概要

Qiitaのトレンド記事をブラウザのツールバーからワンクリックで確認できるChrome拡張機能。

---

## 機能一覧

### 1. トレンド記事一覧表示

ツールバーのアイコンをクリックするとポップアップが開き、Qiitaのトレンド記事を一覧表示する。

**表示項目（1記事あたり）:**

| 表示順 | 項目         | 取得元    | 表示形式                          |
| ------ | ------------ | --------- | --------------------------------- |
| 1      | 記事タイトル | フィード  | テキスト                          |
| 2      | 著者名       | フィード  | テキスト                          |
| 3      | 投稿日時     | フィード  | 相対時刻（下記「投稿日時の表示形式」参照） |
| 4      | LGTM数       | Qiita API | 数値                              |
| 5      | タグ         | Qiita API | タグ一覧                          |

**表示件数:** 上位10件（フィードから取得可能な30件のうち、N+1対策として上位10件に限定）

### 2. トークン登録

Qiita APIの認証に必要なアクセストークンをユーザーが登録するためのフォームを提供する。

**画面の出し分けルール:**

| トークン状態 | 表示画面             |
| ------------ | -------------------- |
| 未登録       | トークン登録フォーム |
| 登録済み     | トレンド記事一覧     |

---

## データ取得

### トレンド記事の取得

- **エンドポイント:** `https://qiita.com/popular-items/feed`
- **方式:** フィードをパースして記事一覧を取得
- **取得件数:** 最大30件（うち上位10件を使用）
- **取得データ:** 記事ID、記事タイトル、著者名、投稿日時

### 記事詳細の取得（タグ・LGTM数）

- **エンドポイント:** `GET /api/v2/items/:item_id`
- **認証:** Bearer トークン（ユーザー発行）
- **リクエスト数:** 最大10件（表示対象の記事分）
- **取得データ:** タグ一覧、LGTM数

## キャッシュ

| 項目     | 仕様                                                    |
| -------- | ------------------------------------------------------- |
| 保存先   | `chrome.storage.local`                                  |
| 有効期限 | 3時間                                                   |
| 対象     | 取得済みの記事情報（フィード＋API取得分を統合したもの） |

**キャッシュフロー:**

1. ポップアップ表示時にキャッシュを確認
2. キャッシュが存在し有効期限内 → キャッシュから表示
3. キャッシュが存在しないまたは期限切れ → フィード＋APIから再取得し、キャッシュを更新

---

## トークン管理

| 項目         | 仕様                                |
| ------------ | ----------------------------------- |
| 保存先       | `chrome.storage.local`              |
| 発行者       | ユーザー（Qiitaの設定画面から発行） |
| 必要スコープ | `read_qiita`（記事情報の読み取り）  |

## 技術構成

| 項目       | 技術                                             |
| ---------- | ------------------------------------------------ |
| 形式       | Chrome拡張機能（Manifest V3）                    |
| UI         | ポップアップ（ツールバーアイコンクリックで表示） |
| ストレージ | `chrome.storage.local`                           |
| 外部通信   | Fetch API                                        |
